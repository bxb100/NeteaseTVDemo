# Vibefy - tvOS 客户端

代码完全开源，支持以下功能：

- 游客自动登录
- 二维码登录
- 多账号切换
- TopShelf
- 每日推荐（歌单、单曲）
- 歌单、播客、MV 分类
- 音视频播放器
- 歌词滚动
- 最近播放（单曲 声音 歌单 专辑 播客）
- 我的收藏（歌单 专辑 播客）
- 我的喜欢
- 我的歌单
- 我的音乐云盘
- 搜索
